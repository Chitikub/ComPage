อุปกรณ์คอมพิวเตอร์และอุปกรณ์ต่อพ่วง

Output Device

Monitor (หน้าจอ)

แสดงผลภาพจากคอมพิวเตอร์ มีหลายขนาดและเทคโนโลยี

Input Device

Keyboard (คีย์บอร์ด)

อุปกรณ์สำหรับป้อนข้อมูลตัวอักษรและคำสั่งต่างๆ

Input Device

Mouse (เมาส์)

ใช้ควบคุมตัวชี้บนหน้าจอเพื่อเลือกและใช้งานโปรแกรม

Input Device

Microphone (ไมโครโฟน)

อุปกรณ์รับสัญญาณเสียงเพื่อบันทึกหรือใช้สื่อสาร

Input Device

Webcam (เว็บแคม)

กล้องบันทึกวิดีโอสำหรับการประชุมหรือการสตรีมมิ่ง

Output Device

Speakers/Headphones

อุปกรณ์แสดงผลเสียง ทั้งแบบสวมหัวและลำโพง

Output Device

Printer (เครื่องพิมพ์)

ใช้พิมพ์เอกสารหรือรูปภาพจากคอมพิวเตอร์ลงกระดาษ
